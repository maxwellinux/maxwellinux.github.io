** Make sure Pip3 is installed

Generally, Pip3 will be installed automatically when Python3 is installed successfully on your machine.

Use pip3 command on terminal to check it out:

pip3

** What we often do with Pip3

1. upgrade

Upgrade Pip3 itself:

pip3 install --upgrade pip

Upgrade other packages:

pip3 install --upgrade [packages]

2. install

Install packages:

pip3 install [packages]

3. uninstall

Uninstall packages:

pip3 uninstall [packages]

4. list

List all packages installed:

pip3 list

List all packages which are not required by others:

pip3 list --not-required

5. show

Show details of packages:

pip3 show [packages]

** FAQ

1. What is the difference between Pip, Pip2 and Pip3?

Pip2 manages Python2 packages while Pip3 manages Python3 packages. Pip is associated with either Python2 or Python3.

So in Python3 environment, pip3 and pip are both available. And in Python2 environment, pip2 and pip are both available.

they even have the same binary file although they have different paths:

diff /usr/bin/pip /usr/bin/pip3

2. Error: externally-managed-environment

If you meet this error, just add the option --break-system-packages to your command:

pip3 install --break-system-packages [packages]

3. Can we install Python packages with Pacman on Arch-based linux?

Yes, it's able for Pacman to install Python packages now and Pip3 is still availble on managing the Python packages installed by Pacman:

sudo pacman -S python-[packages]
