** What is the Pacman

Pacman is the package manager of Arch linux. It combines a simple binary package format with an easy-to-use build system. The goal of pacman is to make it possible to easily manage packages, whether they are from the official repositories or the user's own builds. Pacman is written in the C programming language and uses the tar format for packaging.

** What we often do with Pacman

1. upgrade

Pacman can update all packages on the system with just one command. The following command synchronizes the repository databases and updates the system's packages

sudo pacman -Syu
sudo pacman -Syyu

The option -y means to download fresh package databases from the server

The option -yy means to force to download fresh package databases from the server

The option -u means to upgrade all installed packages

2. install

Install packages:

sudo pacman -S [packages]

3. remove

Remove packages:

sudo pacman -Rns [packages]

The option -n means to remove a package and its dependencies which are not required by any other installed package

The option -s means to stop creating backup files when removing packages

4. list

List all packages installed:

sudo pacman -Q

List all packages no longer required as dependencies in a quiet way:

sudo pacman -Qdtq

And remove these useless packages:

sudo pacman -Rns $(pacman -Qdtq)

5. clean

Pacman stores its downloaded packages in /var/cache/pacman/pkg/ and does not remove the old or uninstalled versions automatically. So it is necessary to deliberately clean up the cache periodically to prevent the directory to grow indefinitely in size.

sudo paccache -r
