** Preface

According to How to install manjaro linux, we can install the manjaro linux successfully. Then, in order to enhance the functionality of the manjaro linux, here are some of my recommended applications.

** Recommended Applications

1. shell

bash

Install bash by pacman:

sudo pacman -S bash

Enter the dir .config and download the configuration from git@github.com:ethanix/bash.git by git clone:

cd .config
git clone git@github.com:ethanix/bash.git

Create the soft link to .bashrc:

ln -sf ~/.config/bash/bashrc/.bashrc ~/.bashrc

Look through .bashrc for more detailed configurations.

2. terminal

st

Enter the dir .config and download the configuration from git@github.com:ethanix/st.git by git clone:

cd .config
git clone git@github.com:ethanix/st.git

Install the dependencies by pacman:

sudo pacman -S make
sudo pacman -S pkg-config
sudo pacman -S ttf-sourcecodepro-nerd

Enter the dir st and install st by make:

sudo make clean install

Look through config.h for more detailed configurations.

3. text

vim

Install vim by pacman:

sudo pacman -S vim

Download the configuration from git@github.com:ethanix/.vim.git by git clone:

git clone git@github.com:ethanix/.vim.git

Install the dependencies by pacman:

sudo pacman -S xclip

Look through vimrc for more detailed configurations.

neovim

Install neovim by pacman:

sudo pacman -S neovim

Enter the dir .config and download the configuration from git@github.com:ethanix/nvim.git by git clone:

cd .config
git clone git@github.com:ethanix/nvim.git

Look through init.vim for more detailed configurations.

tesseract

Install tesseract by pacman:

sudo pacman -S tesseract

Install language packages by pacman, for example:

sudo pacman -S tesseract-data-eng

Browse ArchWiki Packages for more information of language packages and Tesseract documentation to know more about usage.

4. internet

qBittorrent

Warning: Beacuse of using of P2P and copyright affairs or something like that, Use a VPN to make sure you hide your IP address successfully.

Install qbittorrent by pacman:

sudo pacman -S qbittorrent

Without anything else to do, just enjoy it.
