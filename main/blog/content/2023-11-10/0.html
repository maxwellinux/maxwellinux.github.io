How to install and configure MySQL on Arch-based Linux

** Installation

Install MySQL from the official repository using Pacman:

sudo pacman -S mysql

Verify the installation by running the below command:

mysql --version

Initialize the MariaDB data directory and create the system tables in the MySQL database:

mariadb-install-db --user=mysql --basedir=/usr --datadir=/var/lib/mysql

** Start the MySQL server

Turn on the MySQL server:

sudo systemctl start mysqld

Check the server status:

sudo systemctl status mysqld

** Configuration

Start the MySQL configuration script as the sudo user. Follow the instructions and we’ll be prompted to set our MySQL installation’s security settings.

sudo mysql_secure_installation

Now, log in with the command:

mysql -u root -p

or

sudo mysql

Create a new user:

CREATE USER ''@'localhost' IDENTIFIED BY '';

Grant all privileges to the new user:

GRANT ALL PRIVILEGES ON *.* TO ‘’@’localhost’ WITH GRANT OPTION;

Free up the cached memory and exit the MySQL client:

FLUSH PRIVILEGES;
exit;

Log in with the new user:

mysql -u [new user] -p
