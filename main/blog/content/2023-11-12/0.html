** What is the Pacman

Pacman is the package manager of Arch linux. It combines a simple binary package format with an easy-to-use build system. The goal of pacman is to make it possible to easily manage packages, whether they are from the official repositories or the user's own builds. Pacman is written in the C programming language and uses the tar format for packaging.

1. upgrade

Pacman can update all packages on the system with just one command. The following command synchronizes the repository databases and updates the system's packages

sudo pacman -Syu
sudo pacman -Syyu

The option -y means to download fresh package databases from the server

The option -yy means to force to download fresh package databases from the server

The option -u means to upgrade all installed packages
